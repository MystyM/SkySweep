### Purpose
A sweeping data acquisition program designed to present current hotspots of activity among publicly traded companies, conglomerates, and ETF members.

### Tech Stack
- Python
    - Data Processing 
        - PyArrow
        - PySpark
        - Parquet
        - DeltaLake
    - NetworkX (Graphs)
- Scala
